AWS Product Categories

Analytics

AWS has a wide range of services for analytics such as Amazon Athena which helps you to make instant data queries with results in seconds or Amazon Kinesis that enables you to do real-time analysis of video and data streams.

Application Integration

Amazon offers application integration products so that you can integrate serverless applications and distributed systems with ease. Some examples are Amazon Simple Notification Service (SNS) which is a fully managed messaging service, and Amazon EventBridge, a serverless event bus that takes care of event ingestion and delivery, security, authorization, and error handling.

AR & VR

As augmented reality (AR) and virtual reality (VR) capabilities increase, AWS introduces Amazon Sumerian to help you build real-world AR, VR, and 3D applications rapidly with minimal domain-specific expertise.

Blockchain

AWS is a highly trusted cloud service provider for blockchain and ledger technology workloads of many enterprises. Purpose-built products of AWS Blockchain open up many possibilities such as a centralized ledger database with Amazon Quantum Ledger Database (QLDB) or a multi-party, fully managed blockchain network with Amazon Managed Blockchain.

Business Applications

More and more organizations of all sizes use the cloud for available, flexible and agile services. AWS provides multiple solutions for companies to run their software on the cloud such as Amazon WorkMail, a secure, managed business email and calendar service, or Amazon WorkDocs, an enterprise document storage and sharing service.

Compute

AWS Compute products provide processing power to allow you to carry out computational abilities via a series of instructions used by applications and systems. A wide range of AWS Compute products enables you to have virtual service in the cloud (by Amazon EC2) or run your code without thinking about servers (by Amazon Lambda).

Containers

AWS is a very popular choice for many customers to run containers in a secure, reliable and scalable way. In fact, Amazon claims that 80% of all containers in the cloud run on AWS. One advantage of AWS Containers is that you get to choose which container orchestrator to have: Amazon Elastic Container Service (ECS) or Amazon Elastic Kubernetes Service (EKS).

AWS Cost Management

AWS Cost Management Solutions bring instruments to manage your resources by taking your specific needs into account. Your cost & usage data can be analyzed and visualized and precisely chargeback to the right unit. As an example, one of the products in this category, AWS Budgets offers flexible budgeting and forecasting to improve your planning and control your costs.

Customer Enablement

AWS Customer Enablement services are designed to increase your migration speed and other cloud related team capabilities. By using these AWS services such as AWS Managed Services or AWS IQ, you can leverage highly experienced experts in the cloud industry for your particular needs such as starting a project, optimizing costs or more.

Database

Amazon provides one of the broadest selection of purpose-built databases where you can choose from 15+ database engines. Relational databases like Amazon Aurora for traditional applications, or key-value databases like Amazon DynamoDB for high-traffic web apps or e-commerce systems are only a part of the wide selection of AWS Database solutions.

Developer Tools

A wide range of tools for developers are available in the AWS Developer Tools category. These tools help you to boost your software development speed and integrate with AWS with ease. Building, testing and deploying become easier with services such as AWS CodeDeploy which provides automated deployments and uptime management solutions, or AWS Device Farm that supports testing IOS, Android, and web apps.

End-User Computing

As the world becomes more distributed with remote work, organizations require more agile solutions to the dynamic needs of the human resource. Therefore AWS offers End-User Computing (EUC) products for you to manage the high demand flexibility of your users. Amazon Workspaces for desktop-as-a-service or Amazon Work-link for secure mobile access are good examples to try out.

Game Tech

The majority of the worlds biggest public game companies are using Amazon Web Services. AWS has also a product category to accommodate game-tech specific needs such as Amazon GameLift for dedicated cloud servers to boost multiplayer experiences or Amazon Lumberyard, a game engine without any seat fees.

Internet of Things

As the IoT industry is expanding rapidly, billions of devices all around the world provide countless possibilities to explore the full potential of IoT. AWS offers various IoT related services such as AWS IoT Core for smooth connection between devices and cloud applications, or AWS IoT Events for effortless detection and response to IoT events, and many others.

Machine Learning

AWS has a very ambitious goal of enabling every developer with machine learning. With a broad group of ML services supported by the cloud infrastructure, Amazon is definitely one of the key solution providers in this field with countless possibilities. For example, the flagship service of this category, Amazon SageMaker, provides a comprehensive package where data scientists and developers can use from the data preparation phase until the deployment of high-quality machine learning (ML) models.

Management & Governance

AWS Management and Governance products bring a possibility where companies dont need to do trade-offs between innovation and control over costs and security. Services such as AWS Control Tower provides built-in governance control of multi-account AWS environments, or AWS CloudFormation provides provision resources and applications that meet your policies.

Media Services

AWS has a very flexible set of services for live and on-demand video workflows. Integrated with AWS storage, networking, and compute services; you can scale up globally without thinking about maintaining video hardware. Build engaging live streaming with Amazon Interactive Video Service, or let Amazon Kinesis Video Streams capture, process, and store video streams for analytics and machine learning.

Migration & Transfer

Amazon is aiming to make migration as smooth as possible to get the business outcome quickly. Bringing many years of experience, AWS provides a comprehensive set of services such as AWS Database Migration Service to migrate your databases to AWS with minimal downtime, and CloudEndure Migration to automate your mass migration to the AWS cloud.

Mobile

Front-End Web & Mobile services on AWS provide various options for developers to build, test and deploy an application with minimum friction. Products such as AWS Amplify bring the possibility to scale up in an easy, secure and reliable way. As developers can utilize existing iOS/Android IDEs and web frameworks, those products also accelerate the development process.

Networking & Content Delivery

AWS Networking and Content Delivery services aim to make sure that any kind of workload you have runs in the cloud without any problem. AWS is a feasible option if the security, availability and performance consistency is important for your networking services. Among many other things, you can use Elastic Load Balancing to scale your network design, or AWS VPC (Virtual Private Cloud) for isolated cloud resources.

Quantum Technologies

Research shows that quantum computers have the potential to solve some of the impossible or very difficult problems for classical computers. To provide a development environment for quantum researchers and developers, AWS introduces Amazon Braket.

Robotics

As robots are becoming more popular in many fields, they get more complex with many sophisticated functions. AWS RoboMaker in this category provides a complete solution for simulating, testing and deploying robotic applications.

Satellite

Satellites are becoming more crucial in many use cases such as surface imaging, communications and weather forecasting. Amazon created AWS Ground Station as a fully managed cloud service where you can control data from satellites and process it without building a ground station infrastructure.

Security, Identity, & Compliance

To secure your workload in the cloud, AWS offer various solutions in different sub categories such as data protection (e.g. AWS Key Management Service), infrastructure protection (e.g. AWS Shield for DDoS protection), detection (e.g. Amazon Inspector for analyzing application security) and many others.

Storage

AWS provides several services for organizations that are looking for object, file and block storage as well as backup, data transfer and edge computing services. Amazon Simple Storage Service (Amazon S3), one of the popular AWS services, is used for object storage with high levels of security, availability and performance. Other storage services such as AWS Backup and AWS Storage Gateway upgrade organizations' storage capabilities to the next level.
